# merge_video_audio.bat


# FFmpeg Video + Audio Merger

Простой BAT-скрипт для Windows, который помогает объединить отдельный видеофайл и отдельный аудиофайл в один итоговый `.mp4`.

## Возможности

- Проверяет наличие `ffmpeg` в системе.
- Если `ffmpeg` не найден — пытается установить его через `winget`.
- Анализирует текущую папку или папку, переданную скрипту.
- Находит видеофайлы и аудиофайлы.
- Показывает список найденных видео и аудио.
- Позволяет выбрать нужное видео и нужную аудиодорожку.
- Объединяет выбранные дорожки в итоговый `.mp4`.
- При ошибке пробует безопасный вариант с перекодированием.

## Поддерживаемые форматы

### Видео

```text
.mp4, .mkv, .mov, .avi, .webm, .m4v, .mpg, .mpeg, .ts
```

### Аудио

```text
.mp3, .wav, .m4a, .aac, .flac, .ogg, .opus, .wma
```

## Как использовать

1. Скачайте файл `merge_video_audio.bat`.
2. Положите его в папку, где находятся видео и аудио.
3. Запустите двойным кликом.
4. Выберите номер видеофайла.
5. Выберите номер аудиофайла.
6. Дождитесь создания итогового файла.

## Использование с другой папкой

Можно передать путь к папке как аргумент:

```bat
merge_video_audio.bat "D:\Media\Project"
```

Также можно перетащить папку на BAT-файл — скрипт проанализирует её содержимое.

## Итоговый файл

Итоговый файл создаётся рядом с исходными файлами в формате:

```text
имя_видео__plus__имя_аудио.mp4
```

Если такой файл уже существует, скрипт спросит разрешение на перезапись.

## Требования

- Windows 10 / Windows 11
- `winget` для автоматической установки FFmpeg
- `ffmpeg`

Если `winget` отсутствует, FFmpeg нужно установить вручную.

## Примечание

По умолчанию скрипт старается не перекодировать видео:

```text
-c:v copy
```

Аудио конвертируется в AAC для лучшей совместимости с `.mp4`.

Если первый вариант не сработает, скрипт автоматически попробует безопасный режим с перекодированием видео через `libx264`.
